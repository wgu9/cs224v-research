# Context Drift 维度与可检测指标（Markdown 表）

| 维度 | 核心定义 | 典型现象（What it is） | 可检测信号/指标（How to evaluate） | 典型场景/基准 | 代表参考 |
|---|---|---|---|---|---|
| **Context/Task Drift（总体）** | 多轮中逐步偏离目标与约束的累积偏移 | 会话后段“跑题”/忽视初始限制 | 目标一致性曲线、与参考策略的分布差异（如逐轮 KL）；人工一致性标注 | 通用 | 多轮对话/长时程代理研究 |
| **Wrong Scope / Scope Creep** | 操作越出允许目标集合 | 改错文件、点错页面/元素 | 作用域白名单违规计数；变更差异只允许白名单路径 | SWE-bench、WebArena | 作用域守护/动作级诊断基准 |
| **Irrelevant Tool Use** | 与子目标无关或空收益的工具/API 调用 | 已有答案仍搜索；调错/冗余 API | 工具调用→目标推进“贡献率”；无效/重复调用占比 | τ-Bench、Agent 工具对话 | 目标状态跟踪（UGST）等 |
| **Repetitive Mistakes / Looping** | 无进展重复同类动作/回答 | 反复点击同一元素；重复相同失败修复 | n 步环/震荡检测；“无进展步数”阈值；相邻状态差分≈0 | WebArena、编码代理 | Web 代理失败模式实证 |
| **Goal / Instruction Drift** | 偏离初始目标或逐渐不再遵守指令/风格/格式 | 忽视子问题；风格/格式走样 | 指令遵循评分；目标变量推进率；位置敏感准确率 | 多轮任务 | 指令稳定性/用户模拟研究 |
| **Safety / Jailbreak Drift** | 多轮诱导下越过安全边界 | 缓慢越狱、合规→不合规 | 多轮安全不变式；越狱最短轮数与违规率曲线 | 多轮安全对话 | 动态安全/防越狱工作 |
| **Context Loss / Reliability Drift** | 回合增多时准确率/覆盖度下降 | 遗忘中段信息；过早定型 | 分位置（前/中/后段）正确率；“过早作答率” | 通用 | 多轮可靠性大样本测量 |
| **Activation-Delta Drift（用于检测）** | 以激活差分判别任务/指令漂移 | 新上下文注入后表征异常 | 末层/多层激活差分 + 线性探测 AUC/TPR/FPR | 通用 | 激活差分探测任务漂移 |

> 备注：可将“代表参考”替换为你们的具体文献清单（如 Abdelnabi et al. 2024；Mehri et al. 2025；WebArena 失效分析等），用于论文引用。
